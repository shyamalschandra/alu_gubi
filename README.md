- **Driver live TUI:** https://shyamalschandra.github.io/alu-gubi-cli/#driver

## Charter (eleven pillars)

1. Find out all minimum and maximum cost of living (CoL) for all zipcodes in USA.
2. Adversity Score using Bindi to determine the actual CoL.
3. Give the coupons for ALU GUBI to prevent usage for excise goods (alcohol, drugs, and smoking).
4. Unlock stuff with Bindi protocol to make life more enjoyable depending lossless input and output.
5. Earn hotel and vacations depending the work achieved without any disturbance.
6. Personalized gifts from friends that are NFTed and generated by AI.
7. Universal library and law.
8. Point systems for healthcare exercise (gym), dance, biking (or spinning), education (online & personalized), discounts on new books, library (Kindle & Audible credits), rental for textbooks, lifelong learning, and VCs for startup funding.
9. Book-It-like competition that is gamified to give back to the goodreads and comments section of Amazon and other bookstores.
10. Elections that are gamified and given coupons to do advertising so the market is fair and equal for all.
11. Citizen policing, government, and military for ultimate autocracy while meeting CoL.

Now, let's do it!

## Install the industrial CLI (binary only)

```bash
brew tap shyamalschandra/alu-gubi-cli
brew trust --formula shyamalschandra/alu-gubi-cli/alu-gubi-cli
brew install alu-gubi-cli
brew trust --formula shyamalschandra/alu-gubi-cli/alu-gubi-driver
brew install alu-gubi-driver
```

```bash
# Terminal 1
alu-gubi-cli serve

# Terminal 2 — continuous ratatui tour (q to quit)
alu-gubi-driver live --zip 94102 --display-name "Demo Citizen"
```
